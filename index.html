HTML

Cuando hablamos de HTML (Lenguaje de Marcado de Hipertexto), nos referimos a la base estructural de cualquier pagina web. Es el lenguaje que organiza y da forma a los contenidos, permitiendo que los navegadores interpreten y presenten información de manera ordenada. A través de etiquetas, HTML permite definir títulos, párrafos, imágenes, enlaces, tablas, formularios y otros elementos esenciales que conforman una página web. Aunque muchas veces se lo menciona como un "lenguaje de programación", en realidad HTML es un lenguaje de marcado, lo que significa que su función principal no es ejecutar acciones ni operaciones lógicas, sino establecer la estructura del contenido. En otras palabras, actúa como los cimientos y el esqueleto de un sitio web, sobre el cual se apoyan otros lenguajes como CSS (para el diseño visual) y JavaScript (para la interactividad). Sin HTML, no habría una forma coherente de presentar la información en la web tal como la conocemos hoy.

Dentro de este lenguaje podemos encontrar etiquetas que son las iniciales y principales a la hora de crear una pagina web

<html>, <head>, <body>

El desarrollo web moderno no sería posible sin HTML. Gracias a este lenguaje, los desarrolladores pueden definir jerarquías de contenido, incluir imágenes y videos, y enlazar distintas secciones dentro de un sitio o hacia otras páginas. Además, HTML ha evolucionado con el tiempo: la versión actual, HTML5, introdujo mejoras importantes como la inclusión de elementos semánticos, soporte para multimedia sin plugins y mayor accesibilidad. Este lenguaje, aunque simple en su sintaxis, es extremadamente poderoso cuando se combina con otras tecnologías. Su aprendizaje es considerado el primer paso para quienes desean comenzar en el mundo del desarrollo web, ya que permite construir sitios funcionales desde cero y entender la lógica detrás de cómo se presenta la información en internet.

CSS

CSS (Cascading Style Sheets, u Hojas de Estilo en Cascada) es el lenguaje que se utiliza para definir la apariencia visual de los elementos en una página web. Mientras que HTML se encarga de la estructura y el contenido, CSS determina cómo se ve ese contenido: sus colores, tamaños, posiciones, márgenes, tipos de letra, fondos, animaciones y mucho más. El nombre "en cascada" se refiere a cómo las reglas de estilo se aplican de forma jerárquica y prioritaria según el orden en que están definidas. Esto significa que diferentes reglas pueden afectar al mismo elemento, y CSS tiene un sistema que determina cuál se aplica finalmente. Gracias a CSS, un mismo HTML puede adoptar estilos completamente distintos, permitiendo una personalización casi ilimitada del diseño web. Además, CSS permite separar completamente el contenido de la presentación, lo que facilita el mantenimiento de los sitios, mejora la accesibilidad y optimiza la velocidad de carga. En conjunto con HTML y JavaScript, CSS es una de las tecnologías fundamentales de la web.

La forma en la que conectamos HTML con CSS es a través de selectores, que permiten aplicar estilos a elementos específicos del documento. Algunos de los más comunes son:

class, id.

CSS ha evolucionado significativamente desde sus primeras versiones. Hoy en día, gracias a herramientas modernas como Flexbox, Grid, media queries y variables CSS, los diseñadores web pueden crear interfaces responsivas que se adaptan a distintos dispositivos, desde computadoras de escritorio hasta celulares. También es posible aplicar efectos visuales avanzados, como transiciones suaves, sombras, filtros, y animaciones sin necesidad de JavaScript. Aprender CSS no solo significa entender cómo aplicar color o tipografía, sino también dominar el arte de crear experiencias visuales coherentes, accesibles y atractivas para los usuarios.

SQL

SQL (Structured Query Language) es un lenguaje estándar utilizado para gestionar y manipular bases de datos relacionales. que nos permite a los usuarios realizar operaciones como insertar, consultar, actualizar y eliminar datos. Es un lenguaje accesible para principiantes pero también potente para tareas complejas. Si bien es un lenguaje de programacion, se lo considera un lenguaje de programacion de tipo especifico,
Sql es esencial para trabajar con datos en la mayoria de sus aplicaciones, y para la empresas dado que sirve para crear y administrar bases de datos relacionales y llevar un seguimiento de estas, algunas de sus caracteristicas mas importantes son:

⚪Acceso a datos: SQL permite acceder a datos almacenados en bases de datos de forma eficiente y precisa.

⚪Manipulación de datos: Permite realizar modificaciones en los datos, como actualizaciones, inserciones y eliminaciones.

⚪Gestión de bases de datos: SQL es la herramienta principal para administrar la estructura de las bases de datos, crear tablas y definir relaciones entre ellas.

SQL ha evolucionado desde su creación para convertirse en una herramienta fundamental en el manejo de bases de datos. Hoy en día, gracias a sus capacidades avanzadas, permite no solo consultar información, sino también organizar, filtrar, actualizar y proteger grandes volúmenes de datos de manera eficiente. Con SQL, los desarrolladores y analistas pueden realizar operaciones complejas como uniones entre tablas, agrupamientos, subconsultas, procedimientos almacenados y manejo de transacciones, lo que lo convierte en un lenguaje clave para el análisis de datos y la programación backend. Aprender SQL no solo implica saber hacer consultas básicas, sino también comprender cómo estructurar bases de datos de forma óptima, mantener su integridad y extraer información útil para la toma de decisiones.


SQL se utiliza a través de comandos y sentencias que se ejecutan en un sistema de gestión de bases de datos (DBMS). Los DBMS, como MySQL, PostgreSQL, SQL Server, Oracle, etc., interpretan las sentencias SQL y realizan las operaciones correspondientes en la base de datos.

C++

C++ es un lenguaje de programación de propósito general, de alto nivel y compilado, conocido por su versatilidad y eficiencia. Se utiliza para desarrollar una amplia gama de aplicaciones, desde sistemas operativos y videojuegos hasta software de alto rendimiento y aplicaciones científicas,

Algunas de sus caracteristicas son

⚪Multiparadigma: Permite programación orientada a objetos, programación estructurada y programación genérica, lo que lo hace muy flexible

⚪Orientado a objetos: Incorpora conceptos como clases, herencia, polimorfismo y encapsulación, lo que facilita la organización y reutilización del código.

⚪Alto rendimiento: Permite un control cercano del hardware y la memoria, lo que lo hace ideal para aplicaciones que requieren velocidad y eficiencia.

C++ es un lenguaje de programación poderoso y versátil, utilizado ampliamente en el desarrollo de software de alto rendimiento. Desde sus inicios, ha sido la base para crear sistemas operativos, videojuegos, motores gráficos, software embebido y aplicaciones que requieren control preciso del hardware y la memoria. A lo largo del tiempo, C++ ha incorporado características modernas como la programación orientada a objetos, plantillas (templates), manejo de excepciones y, en sus versiones más recientes, mejoras en la sintaxis y soporte para programación funcional. Aprender C++ no solo significa escribir código que funcione, sino también comprender cómo se gestionan los recursos del sistema, optimizar el rendimiento y construir soluciones robustas, eficientes y escalables..
